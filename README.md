# IMPLUS — Система симуляции разума

Изолированная симуляция чистого разума. Разум существует в собственном временном измерении, начинает с минимальных концепций и растёт через намеренные вводы. Никаких эмоций, тела, социального контекста — только чистое рассуждение и накопленные концепции.

## Архитектура

| Компонент | Технология |
|-----------|-----------|
| Backend   | Python 3.11+ · FastAPI · SQLite |
| Frontend  | React 18 · TypeScript · Vite · Tailwind CSS |
| Граф      | networkx · react-force-graph-2d |
| LLM       | Claude API (`claude-sonnet-4-6`) |

## Временная система

- Коэффициент **6:1** — 1 реальная минута = 6 минут разума
- 10 реальных минут = 1 час разума
- 4 реальных часа = 1 день разума
- Время рождения разума (`born_at`) хранится в БД **навсегда** и никогда не сбрасывается
- Даже после перезапуска сервера разум продолжает с того момента, где остановился

## Быстрый старт

### 1. Клонировать и настроить

```bash
cp .env.example .env
# Отредактируйте .env:
#   IMP_PASSWORD=ваш_пароль
#   ANTHROPIC_API_KEY=sk-ant-...
#   SESSION_SECRET=длинная_случайная_строка
```

### 2. Backend

```bash
cd backend
pip install -r ../requirements.txt
uvicorn main:app --reload --port 8000
```

При первом запуске:
- Создаётся БД `data/mind.db`
- Фиксируется время рождения разума
- Засеваются 6 базовых концепций

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
# Открыть http://localhost:5173
```

## Три вида

### ПОТОК
Главная обсерватория. Слева — часы разума и живой поток мыслей. Справа — интерактивный граф концепций. Мысли цветокодированы: серые (спонтанные), синие (реакции), золотые (рубежи), белые (созерцания).

### БИБЛИОТЕКА
Все концепции в виде карточек, отсортированные по времени добавления или по количеству связей. Разворачиваемый журнал обработки показывает, как каждая концепция изменила граф.

### СОЗЕРЦАНИЕ
Ввод мысли для анализа или добавление новой концепции в граф. Ответ разума стримится в реальном времени.

## Семенные концепции

При первом запуске разум знает только:

- `есть/нет` — существование и отсутствие
- `я/не-я` — граница себя
- `до/после` — порядок во времени
- `одинаково/различно` — отношение сходства
- `больше/меньше` — отношение величин
- `здесь/не-здесь` — пространственное различие

## Рубежи разума

| Рубеж | Реальное время | Время разума |
|-------|---------------|-------------|
| Первый час | 10 минут | 1 час |
| 6 часов | 1 час | 6 часов |
| Первый день | 4 часа | 24 часа |
| Первая неделя | 28 часов | 7 дней |
| Первый месяц | ~5 дней | 30 дней |
